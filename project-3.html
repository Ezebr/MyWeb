云原生设备管理平台

2021.05 - 2022.03
Vue 3 Node.js Docker

项目背景

面对分布在全国各地的百万级终端设备，传统的管理后台在数据量级庞大时面临崩溃、响应迟缓以及无法直观展示空间数据的问题。我们需要构建一个具备高扩展性的大型云原生可视化平台。

核心挑战

解决前端海量 DOM 节点渲染导致的性能瓶颈；实现服务端针对百万级设备并发状态更新的毫秒级推送引擎。

解决方案与架构

前后端彻底分离，全面拥抱云原生架构。后端基于 Node.js 微服务化，前端使用 Vue 3 结合 WebGL 技术构建 3D 可视化大屏。

  • 微服务架构： 将设备接入、规则引擎、报警模块拆分为独立服务，通过 Docker 与 Kubernetes 实现自动化扩缩容。
  • 千万级数据推送： 基于 Redis Pub/Sub 与 WebSocket 集群，实现海量设备状态变更的实时下发。
  • 3D 数据可视化： 引入 Three.js，通过解析 CAD 图纸在浏览器中重构工厂三维模型，实现报警节点的空间定位。

落地成果

100W+ 设备并发接入
60FPS 大屏渲染帧率
0.5s 报警响应时间
