# 智慧信号显示器

# 摘要

本项目设计并实现了一款基于国产安路 EG4S20BG256 FPGA的高性能、低成本数字存储示波器、信号发生器与语音控制一体化系统。示波显示部分通过高速模数转换器（ADC）完成数字化采样；信号发生输出部分则采用高精度数模转换器（DAC），实现多种波形的生成，并具备上位机通过 WiFi 或蓝牙模块向核心板发送信号进行控制的功能。FPGA作为核心处理单元，负责高速数据采集、触发判断、波形存储、显示驱动及信号生成控制，充分发挥其并行处理能力，确保系统的实时性。人机交互部分集成了语音控制与上位机控制功能，用于调节波形幅值、频率等参数，并通过 VGA接口实现波形与参数的图形化显示。测试结果表明，本作品示波器部分可实现最高 50MS/s 的实时采样率，具备波形参数自动测量与存储功能；信号发生器部分可输出正弦波、方波、三角波三种常用波形，频率范围为1KHz 至 60MHz，幅度可调。该系统集测量、生成与便利化于一体，适用于教育实验、工程调试与科研开发，为 FPGA在混合信号系统中的自主应用提供了完整的案例。

# 1.1 功能与特性

本作品集成了数字存储示波器和信号发生器双重功能，并兼具上位机控制功能，展现出高集成度与卓越的灵活性。示波器部分：支持双通道同步采集，最高采样率可达 50MS/s，具备自动测量、波形存储等实用功能。信号发生器部分：能够输出多种标准波形，并精确调节频率与幅度。上位机控制方面：上位机可通过WiFi 或蓝牙模块发送控制信号，灵活修改波形、幅值等参数。

系统以 FPGA为核心，实现了数据采集、信号生成、显示与控制的全流程硬件加速，确保了低延迟和高稳定性。用户可通过按键进行便捷操作，VGA 界面实时显示波形与相关参数，整体设计紧凑高效，成本控制合理。

# 1.2 应用领域

在电子电路设计与调试过程中，工程师通过使用信号发生器向放大器注入测试信号，并借助示波器观察输出波形，从而评估其增益、带宽和线性度。在通信系统研发领域，信号发生器用于产生标准的调制信号，以测试接收机的灵敏度，而示波器则负责分析信号的调制质量与完整性。在汽车电子领域，信号发生器能够模拟传感器信号，示波器则用于捕获和解码复杂的 CAN总线通信数据，以验证电子控制单元的响应性能。

此外，无论是验证音频设备的性能，测量电源模块的纹波噪声，还是在科研实验中捕捉物理现象，或是进行生产线上产品的最终测试，示波器与信号发生器这对组合都发挥着不可或缺的重要作用。总而言之，示波器与信号发生器相辅相成，前者负责解读电路的“回答”，后者则负责提出精确的“问题”，它们共同构成了连接理论设计与物理现实、确保电子产品从概念到成品全过程可靠性与性能的基石。

# 1.3 主要技术特点

1. 高度集成化设计：在同一 FPGA 平台上实现了数字存储示波器与信号发生器的双功能集成，系统结构紧凑，成本可控。

2. 高性能数据采集：采用 14 位双通道 ADC，最高采样率可达 50MS/s。

3. 高精度信号生成：配备 14位DAC，支持正弦波、方波、三角波输出，最高频率达 60MHz。

4. 实时波形处理：FPGA实现高速数据缓存、触发判断、波形存储及显示驱动，确保系统的实时性与稳定性。

5. 多样化人机交互：支持按键与语音控制，同时上位机可通过蓝牙或 WiFi模块进行控制，实时调节波形参数，并通过 VGA接口实现图形化界面显示。

6. 模块化硬件设计：ADC 与DAC 模块、蓝牙与 WiFi 模块独立设计，便于系统扩展与维护。

# 1.5 主要创新点

1. 双功能一体化设计：在同一 FPGA 平台上集成示波器与信号发生器，实现信号采集与生成的协同运作，显著提升了仪器的实用性和性价比。

2. 语音与上位机协同控制设计：在示波器或信号发生器功能运行过程中，可通过语音指令或上位机操作，灵活调节示波器或信号发生器的各项参数。

3. 软硬件协同优化：借助 FPGA 技术实现采集、显示、控制的全流程硬件

加速，并结合语音识别与 VGA 显示功能，构建高效的人机交互闭环系统。

# 1.6 设计流程

本系统设计采用自顶向下的方法，首先明确系统功能需求，制定整体架构。随后进行硬件选型与电路设计，涵盖 ADC/DAC 模块、FPGA 最小系统、蓝牙及WiFi 模块。在 FPGA 中，采用模块化设计，分别实现数据采集、信号生成、显示驱动与语音控制等功能。最后，进行系统联调与性能测试，优化算法及界面交互。整个设计流程清晰，各阶段任务明确，确保了项目的高效推进与最终实现。

智能信号显示器的模块化结构，分为三个主要模块：控制模块、信号发生器模块和示波器模块。

# 1. 控制模块

智能语音模块：该模块的核心功能是接收用户的语音指令，并通过内置的语音识别技术将其转换为可执行的控制命令。通过对天问语音模块进行编程，可以预设一系列的控制命令和对应的引脚输出数据。当模块识别到特定的语音指令时，它将自动发送相应的数据到指定的引脚。当 FPGA 开发板接收到智能语音模块发送的控制数据时，它将根据预设的程序逻辑执行相应的功能操作。这种交互确保了系统的响应性和灵活性，使得用户可以通过语音指令直接控制信号显示器的各种功能。

WiFi 模块：该模块提供无线网络连接功能，允许智能信号显示器通过局域网与手机、电脑等设备上的上位机程序进行通信。通过 WiFi 模块，用户可以在局域网内远程控制 FPGA 开发板，发送指定的控制命令，实现对信号显示器的远程操作和管理。

按键模块：智能信号显示器系统中提供用户直接物理交互的界面，它允许用户通过物理按键对信号进行直接修改和控制。按键模块可以与 SW 拨码开关组合使用，实现对信号各项参数的更精细控制。这种组合提供了用户对信号特性的广泛调节能力，以满足不同的测试和应用需求。为了减少误操作，按键模块还包含防抖设计，维持稳定性与安全性。

按键消抖：用于消除按键操作时的抖动，确保信号稳定。

# 2. 信号发生模块

频率控制模块：用于调整信号的频率。通过频率控制逻辑，通过逐步增加或重置频率控制字来改变输出信号的频率，以实现对输出信号频率的精确控制。

波形改变模块：这个模块通过按键和拨码开关提供了用户友好的界面，允许用户选择不同的波形输出，并可以精细控制输出波形的幅度。通过语音控制信号，还可以实现波形和幅度的语音控制，增加了模块的交互性。这种设计在数字信号处理和波形发生器应用中非常常见，提供了灵活的波形生成和控制能力。

振幅改变模块：调整输出信号的振幅。用户通过物理按键或语音指令来精确控制输出信号的频率。这种设计在数字信号发生器中非常实用，因为它提供了一种简单而有效的方式来调整信号的频率。

相位改变模块：改变信号的相位。通过按键、拨码开关和语音指令提供了用户友好的界面，允许用户选择不同的相位输出，并可以精细控制输出信号的相位。通过语音控制信号，还可以实现相位的语音控制，增加了模块的交互性。

# 3. 示波器部分

VGA 显示模块：通过 VGA 接口显示信号波形。是数字信号处理系统中的一个重要组成部分。通过生成同步信号和数据信号，VGA 模块可以实现

对波形数据和各项参数信息的显示。

参数计算模块：智慧信号处理系统中的一个重要组成部分，它提供了对输入信号特性的详细分析。通过计算峰峰值、最大值、最小值、占空比和频率，该模块可以帮助用户更好地理解和描述信号的特性。

# 第四部分 总结

# 4.1 可扩展之处

本作品在未来仍有显著的提升空间。硬件上，可升级至更高性能 ADC/DAC芯片，并采用更高速的 FPGA（如安路PH系列）以处理随之增长的数据吞吐量。软件上，可引入软核处理器（如 RISC-V），将用户界面、文件系统、通信协议等复杂任务交由处理器处理，而 FPGA逻辑专注于高速采集，形成高效的软硬协同架构。功能上，可增加数学运算功能（FFT 频谱分析、滤波器）、高级触发模式（脉宽、斜率、串行协议触发），从而将其从一款基础示波器升级为混合信号测试平台。

# 4.2 心得体会

本项目的完成，是一次将理论知识转化为工程实践的系统性锻炼，团队成员在技术能力和工程思维方面均获益匪浅。首先，从理论到实践的跨越，深化了我们的技术认知。书本上的数字逻辑、信号处理理论，在项目中具体化为时序约束、资源优化和系统调试等实际操作。通过解决 ADC 数据同步、DDS 相位累积、显示控制等实际问题，我们深刻体会到工程细节对系统性能的决定性影响。

其次，系统级设计思维的建立至关重要。我们认识到，优秀的设计需要统筹全局：FPGA逻辑资源、ADC/DAC 性能、WiFi 和蓝牙模块以及用户界面体验必须协同优化。这培养了我们在多重约束下进行权衡和决策的能力。

再者，调试过程极大提升了我们的问题解决能力。从波形失真到触发不稳定，每一次故障排查都是一次严谨的逻辑训练。我们学会了综合利用仿真、仪器测量和代码分析来定位问题根源，锻炼了面对复杂系统的耐心与洞察力。

最后，团队协作是项目成功的关键。硬件设计与软件开发的紧密配合，让我们深刻体会到跨领域沟通的重要性。通过明确分工、定期同步和知识共享，我们高效地克服了技术难点，确保了项目的顺利推进。

此次经历不仅让我们掌握了 FPGA系统开发的全流程，更培养了严谨的工程态度和团队合作精神，为未来的技术探索奠定了坚实基础。
