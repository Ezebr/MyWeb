智能家居系统

2024 年 3 月
ESP32 MQTT React

项目简介

这是一个基于 ESP32 的智能家居控制系统，用户可以通过手机 App 远程控制家中的各种设备，如灯光、窗帘、空调等。系统支持自动化场景设置，可以根据时间、传感器数据等条件自动执行预设操作。

主要功能

  • 远程控制：通过 MQTT 协议实现设备的远程控制
  • 传感器数据采集：温度、湿度、光照等环境数据实时监测
  • 自动化场景：支持基于时间和传感器数据的自动化控制
  • 设备管理：设备添加、删除、分组管理
  • 数据可视化：历史数据图表展示

技术架构

硬件层

使用 ESP32 作为主控芯片，连接各种传感器和执行器。ESP32 具有 WiFi 和蓝牙功能，可以方便地接入网络。

通信层

采用 MQTT 协议进行设备与服务器之间的通信。MQTT 是一种轻量级的消息传输协议，非常适合物联网场景。

服务层

后端使用 Go 语言开发，负责设备管理、消息转发、数据存储等功能。数据库使用 MySQL 存储设备信息和历史数据。

应用层

前端使用 React 开发，提供友好的用户界面。支持 Web 和移动端访问。

遇到的挑战与解决方案

挑战 1：设备稳定性

ESP32 在长时间运行后偶尔会出现掉线问题。通过添加看门狗机制和自动重连逻辑解决了这个问题。

挑战 2：消息延迟

在设备数量较多时，MQTT 消息出现延迟。通过优化 QoS 等级和使用消息队列解决了这个问题。
