智能家居中枢控制系统

2023.08 - 2024.01
ESP32 MQTT React

项目背景

随着物联网技术的普及，传统智能家居方案存在响应延迟高、数据隐私堪忧（强依赖公有云）等问题。本项目旨在打造一个完全基于本地局域网的高性能中枢控制系统，确保毫秒级的控制响应，同时实现数据的绝对安全与本地化存储。

核心挑战

在资源受限的微控制器（MCU）上实现高并发的 WebSocket 连接与稳定不掉线的 MQTT 报文处理，同时需保证系统在断网情况下的全功能独立运行。

解决方案与架构

系统采用“边缘中枢+分布节点”的架构。主控采用 ESP32 芯片，运行定制的 FreeRTOS 系统，通过 MQTT 协议与各个子节点进行低延迟通信。

  • 通信层： 引入自适应心跳保活机制，确保 WebSocket 和 MQTT 的长连接稳定性。
  • 控制层： 本地化场景引擎，支持基于时间、传感器状态的多条件触发规则。
  • 交互层： 采用 React 开发的响应式 Web 面板，通过 WebSocket 直接与主控握手，实现无感刷新。

落地成果

15ms 平均指令延迟
99.9% 局域网可用性
50+ 稳定连接节点
