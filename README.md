# Ezebr | 高级开发者作品集

个人作品集网站，展示全栈开发项目与技术能力。

## 项目结构

```
├── index.html          # 主页
├── project-1.html      # 项目详情页 1
├── project-2.html      # 项目详情页 2
├── project-3.html      # 项目详情页 3
├── css/
│   ├── style.css       # 主样式
│   └── project.css     # 项目页样式
├── js/
│   └── main.js         # 主脚本
├── images/             # 图片资源
└── projects/           # 项目资源
```

## 技术栈

- HTML5
- CSS3 (响应式设计)
- JavaScript
- [Phosphor Icons](https://phosphoricons.com/)

## 部署

将文件部署到 Nginx 或任何静态文件服务器即可。
